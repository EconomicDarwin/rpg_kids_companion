# CLAUDE.md — The Hero's Book

Companion PWA for the Lima Clan tabletop campaign. Runs on the girls' Amazon Fire tablets
(Silk browser). It is a viewer and light bookkeeper over `app/data/player_data.json`, not a
way to play the game. Read `README.md` for the full brief; it is accurate and current.

## The two non-negotiable rules

1. **The export is the secrets filter.** Only content the players have discovered at the
   table (gated by canon `01_Campaign_Bible/current_state.md`) may appear in
   `player_data.json` or anything else that ships to a tablet. Hidden mysteries exist in
   canon files and must never leak. When in doubt, leave it out and flag it.
2. **Kid-facing writing style:** short kid-sized sentences, no em-dashes, no semicolons.
   Readers are 7 and 5 (the 5-year-old is learning; this app is her reading incentive).

## Tech constraints

- Plain HTML, CSS, vanilla JS. **No framework, no build step, no dependencies.**
  `app/` is the deployable site as-is.
- `app/data/player_data.json` is generated by `tools/export_player_data.py`
  (see `tools/README.md`). The mock/generated file IS the schema contract with
  `app/js/app.js`. Change shape only with a reason, updating both sides together.
- **Bump `CACHE_VERSION` in `app/sw.js` on every deploy** so tablets pick up changes.
- Hosting is live on Cloudflare Pages: every push to `main` deploys `app/`
  automatically (no build command). Details and troubleshooting: `docs/hosting.md`.
  A push without a cache bump will look like nothing changed on the tablets.

## Run and verify locally

```
cd app
python -m http.server 8080
```

Open http://localhost:8080. A served origin is required (`file://` will not load the JSON
or register the service worker). Verify: all three hero tabs render, Treasure shows party
items, Our Story lists one journal entry per played session. Headless Chrome works for a
smoke check.

## The export

```
python tools/export_player_data.py
```

Reads canon, validates against the kid-text allow-list in `tools/kid_text.json`, writes
`app/data/player_data.json`, and copies needed art from canon `05_Asset_Pipeline/Final_Art/`
into `app/art/`. It fails loudly on any canon entity with no allow-list decision. That
default-closed behavior is the secrets filter. Never weaken it to "skip silently."
